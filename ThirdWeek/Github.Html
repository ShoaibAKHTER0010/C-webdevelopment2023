Git and Github

Git:

git is a version controle system,and it is a tool that help to track changes in code.

Github:

github is a webside that allow user to store and manage their code using git.

Git configure:


some basic git commands

1.clone:

copying the repository from remote to our local machine,"git clone (--link--)".

2.status:

it display the state of code,"git status".

3.cd:

it is used to change directory in code "cd"

4.add:

it add changes in file from working area to the git staging area,"git add (--file name--)".

5.commit:

it is the record of changes,"git commit-m main "massage"".

6.push:

it is used to upload local repo on remote repo"git push origin main".

the init commond 

it is used to create a new git repo"
  1. git init
  2. git remote add origin(--link--)
  3. git remote -v (to verify remote)
  4. git branch
  5. git branch -m main(to rename branch)

git branches

  1. git branch
  2. git branch -m main(to rename branch)
  3. git checkout(branch name) -->to change branch
  4. git checkout -b(new branch name)-->to make new branch
  5. git branch -d (branch name)-->to delete branch.

some types of file

  1. untracked file(new file that is not track by the git)
  2. modified file(the changed in file)
  3. unmodified(no change in file
  4. staged file(file ready to be commited).
