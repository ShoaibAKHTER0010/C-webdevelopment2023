ATTRIBUTES

In computing and computer programming, an attribute is a changeable property or characteristic of some component of a program that can be set to different values. 
 For example, eye color is an attribute of a person, while screen size is an attribute of a smartphone or TV.

1.CORE ATTRIBUTES

Core Attributes Specifies whether the contents of the element are editable.

a.ID: 
The id attribute specifies a unique id for an HTML element(the value must be unique within the HTML document)
. The id attribute is most used to point to a style in a style sheet .

b.CLASS: 
The HTML class attribute specifies one or more class names for an Classes are used by CSS and JavaScript to select and access specific elements.

c.TITLE: 
The title attribute specifies extra information about an element. The information is most often shown as a tooltip text when the mouse moves over the element eg please wait

d.STYLE: 
The style attribute specifies an inline style for an element. The style attribute will override any style set globally, 
e.g. styles specified in the "style" tag or in an external style sheet.

2.INTERNATIONALIZATION ATTRIBUTES

Internationalization. When Internationalization appears in the attribute list, it means the element accepts the set of attributes used to specify language and reading direction

a.DIR: 
The dir attribute specifies the text direction of the element's content. The dir attribute can have the following values
: ltr - means left-to-right text direction. rtl - means right to left text direction.eg 
right to left

b.LANGUAGE: 
The lang attribute specifies the language of the element's content. Common examples are "en" for English, "es" for Spanish, "fr" for French and so on.

c.XML.LANG: 
It is a markup language that is used to store and transport data.it is not a programing language but,it is used to create data structures that can be processed by program.

3.GENERIC ATTRIBUTES:

Vertically aligns tags within an HTML element.

a.ALIGN: 
The align Attribute in HTML is used to specify the alignment of text content of The Element. this attribute is used in all 
well come

b.WIDTH: 
The width attribute specifies the width of the element, in pixels.

c.HEIGHT: 
The height attribute specifies the height of an image, in pixels. Tip: Always specify both the height and width attributes for images.

d.SRC: 
src attribute specifies the URL of the image to use as a submit button.

e.ALT: 
The alt attribute specifies an alternate text for an area, if the image cannot be displayed. 
The alt attribute provides alternative information for an image if a user for some reason cannot view.

f.BGCOLOUR: 
The HTML bgcolor attribute is used to set the background color of an HTML element.

g.BORDER: 
The HTML border attribute is used to set visible border width to most HTML elements within the bordor.
